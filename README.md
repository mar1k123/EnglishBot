# [EnglishBot](https://t.me/nlishBotLearningBot)

Telegram-бот для изучения английских слов на `Aiogram 3` с личным словарем, тренировками, напоминаниями и системой прогресса.


## Основные возможности

- Личный словарь:
  - `/add` - добавить слово (с предложением автоперевода)
  - `/allwords` - показать словарь
  - `/delete_word` - удалить слово

- Режимы тренировки:
  - `/check` - перевод EN -> RU
  - `/check_reverse` - перевод RU -> EN
  - `/check_context` - перевод слова в контексте предложения

- Умная проверка ответов:
  - учитываются альтернативные варианты перевода (не только одна строка из БД)
  - нормализация ответа (регистр, лишние символы, `ё/е`)

- Прогресс и достижения:
  - `/stats` - статистика правильных/ошибок и точность
  - `/hard_words` - сложные слова
  - `/achievements` - уровень и подсказка по росту уровня

- Напоминания:
  - `/reminder` - интерактивная настройка таймера (вкл/выкл, пресеты, свое время)
  - поддержка старого формата `/reminder 15` и `/reminder off`

## Что нового (2026 update)

- Добавлен режим `/check_context`:
  - генерация контекстного предложения для слова
  - выделение целевого слова прямо в тексте
  - проверка ответа через расширенный список переводов

- Добавлена SRS-логика (spaced repetition):
  - интервалы повторения растут при правильных ответах
  - при ошибке слово возвращается в ближайшее повторение
  - работает в `check`, `check_reverse` и `check_context` для личного и общего словаря

- Улучшена работа со сложными словами:
  - слово автоматически уходит из `hard_words` после 10 правильных ответов подряд
  - при новых ошибках может снова стать сложным

- Добавлена универсальная остановка процесса:
  - кнопка `Стоп` в сценариях пошагового ввода
  - корректный выход из FSM-состояний

## Команды

- `/start` - приветствие и список возможностей
- `/add` - добавить слово
- `/allwords` - показать словарь
- `/delete_word` - удалить слово
- `/check` - тренировка EN -> RU
- `/check_reverse` - тренировка RU -> EN
- `/check_context` - тренировка с контекстом
- `/stats` - статистика
- `/hard_words` - сложные слова
- `/achievements` - уровень
- `/reminder` - настройка напоминаний

## Структура проекта

- `main.py` - запуск бота, polling, фоновые задачи
- `handlers.py` - хендлеры команд, FSM-логика, тренировки, напоминания
- `database.py` - SQL-логика, статистика, hard words, SRS-поля и обновления
- `keyboards.py` - Reply-клавиатуры
- `services/` - сервисные модули (пользовательские/общие слова, пути)
- `config.py` - конфигурация бота (токен)

## Быстрый старт

1. Создайте виртуальное окружение:
   - `python -m venv .venv`
2. Установите зависимости:
   - `.venv\Scripts\pip install -r requirements.txt`
3. Укажите токен бота в `config.py`.
4. Запустите:
   - `.venv\Scripts\python main.py`

## Технические заметки

- Используется `SQLite` (локальная БД в корне проекта).
- Контекстные примеры берутся из API словаря с fallback-шаблонами.
- Для проекта настроен `.gitignore` для локальных артефактов (`.venv`, `*.db`, `__pycache__`, временные файлы).
